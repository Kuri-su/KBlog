## 技术类

* Kubernetes 权威指南 - 从 Docker 到 Kubernetes 实践全接触 ( 龚正 吴治辉 王伟 崔秀龙 闫健勇 等著 ) (第二版)
   * 阅读进度 5%
   > 本书在开头快速的过了一遍 kubernetes 中的对象种类, 在 3 章对 Kubernetes的各个组件进行了深入的介绍, 在 4 章对 基于 Kubernetes Api Server 的开发进行了介绍, 在第六章更有 对 Kubernetes 的源码导读, 十分惊喜, 期待收获丰厚!

* 汇编语言 ( 王爽 著 ) (第三版)
   * 阅读进度 10%
   > 本书由浅入深对汇编进行讲解, 且简单易懂, 十分适合入门使用

* Kubernetes In Action 中文版 ( Marko Luksa 著 | 七牛容器云团队 译 )
   * 阅读进度 50% (跳着看)
   * 吸收率: 30%
   > 本书的前半段是对 Kubernetes 中的对象进行遍历讲解, 从 pod / rc / rs / service ..... 进行非常详细的讲解, 作为初学者的学习材料挺合适, 在 11 章开始对 Kubernetes 的机理和架构进行重点讲解, 在 12-17 对 Kubernetes 的高级使用特性进行讲解, 在 18 章 对 CRD 和 operator 的实现进行 Demo 讲解. 本书对单个对象的细致讲解适合作为工具书查阅和入门指引, 但是因为对单个对象的讲解实在篇幅太长了...而且大部分的讲解集中于使用特性, 看久了难免让人感觉有些枯燥.....所以后面就跳着看完了....
